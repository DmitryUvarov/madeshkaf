Для установки gulp в новый проект нужно копировать следующие файлы: #src, gilpfile.js и package.json. 

Далее нужно открыть папку с новым проектом в редакторе кода. А за тем прописать в терминале npm i 

После установки всех пакетов нужно установить converter командой - npm install webp-converter@2.2.3 --save-dev

После всего этого можно запускать gulp
